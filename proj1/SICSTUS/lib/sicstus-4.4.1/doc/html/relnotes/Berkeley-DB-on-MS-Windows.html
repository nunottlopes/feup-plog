10.1 Berkeley DB on MS Windows

When installing Berkeley DB on Windows you should use the binary installer available from Oracle. There are installers available for both 64-bit and 32-bit versions of Berkeley DB, you need the 64-bit installer for 64-bit version of SICStus, and vice versa.

When using Berkeley DB under Windows, you should set the PATH environment variable to contain the path to libdb51.dll. Consult the Berkeley DB documentation for further info.
